# Aiuda - PlantUML Renderer

## Teknikstack

- React 18
- Vite
- TypeScript
- npm (package manager)
- Endast useState för state management

## Installation och körning

```bash
npm install
npm run dev
```

## Utveckling

Projektet använder Vite som build tool och development server. Kör `npm run dev` för att starta utvecklingsservern.
